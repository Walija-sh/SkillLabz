# SkillLabz – Peer-to-Peer Tool Rental & Skill Sharing Platform

SkillLabz is a Final Year Project (FYP) developed at the University of Engineering and Technology (UET) Taxila. It is a web-based peer-to-peer platform that enables users to rent tools and equipment from nearby community members while also providing opportunities for skill sharing.

The platform aims to reduce the cost of purchasing expensive tools for one-time use, encourage better utilization of idle resources, and build trust through secure verification and digital agreements.

## Features

- User Registration & Authentication
- Verified User Profiles
- Tool Listing & Search
- Peer-to-Peer Rental Requests
- Digital Rental Agreements
- OTP-Based Pickup & Return Verification
- Real-Time In-App Chat
- Ratings & Reviews
- Skill Sharing Marketplace
- Owner & Renter Dashboards
- Premium Membership with Boosted Listings
- Real-Time Notifications

## Project Objectives

- Make expensive tools affordable through peer-to-peer rentals.
- Enable owners to monetize idle equipment.
- Build trust using verification, OTP confirmation, and digital agreements.
- Promote learning through integrated skill-sharing.
- Support students, freelancers, and DIY enthusiasts.
